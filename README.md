Sistema de Promedio de Estudiantes en C++
Descripción

Este programa fue hecho para trabajar con estructuras de control en C++. Permite ingresar varios estudiantes, sus calificaciones y calcular el promedio de cada uno.

Dependiendo del promedio, el programa indica si el estudiante aprobó o reprobó.

¿Qué hace el programa?
Pide cuántos estudiantes se van a ingresar
Solicita el nombre de cada estudiante
Pide 4 calificaciones
Calcula el promedio automáticamente
Muestra si aprobó o no
Presenta todo en forma de tabla
¿Cómo funciona?

Se usa un ciclo for para repetir el proceso con cada estudiante.
Luego se calcula el promedio sumando las 4 notas y dividiéndolas entre 4.

Después, con un if, se verifica:

Si el promedio es mayor o igual a 70 → Aprobado
Si es menor → Reprobado
Herramientas usadas
C++
Consola (Visual Studio o cualquier compilador)
